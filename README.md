## Website

## Chức năng hiện có

- Giao diện responsive cho máy tính và điện thoại.
- Tìm kiếm và lọc sản phẩm.
- Giỏ hàng tương tác trên trình duyệt.
- Nội dung quà tặng doanh nghiệp và cẩm nang sức khỏe.
