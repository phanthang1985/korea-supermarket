# Triển khai Korea Supermarket lên GitHub Pages

## 1. Tải mã nguồn lên repository

1. Mở https://github.com/phanthang1985/korea-supermarket
2. Chọn **Add file → Upload files**.
3. Giải nén gói ZIP này trên máy tính.
4. Kéo toàn bộ nội dung bên trong thư mục đã giải nén vào vùng tải lên GitHub. Cần giữ nguyên các thư mục `assets` và `.github`.
5. Chọn **Commit changes** vào nhánh `main`.

## 2. Bật GitHub Pages

1. Mở **Settings → Pages** trong repository.
2. Tại **Build and deployment → Source**, chọn **GitHub Actions**.
3. Mở tab **Actions** và chờ workflow `Deploy GitHub Pages` hoàn thành.

Website dự kiến:

https://phanthang1985.github.io/korea-supermarket/
